O que é o SisInfo

Última atualização: 18 de Março de 2026 • Tempo de leitura: 5 min

lightbulb

Visão Geral

O SisInfo (Sistema de Informações) é uma plataforma Web GovTech focada em análise de dados, inteligência territorial e indicadores socioeconômicos para municípios brasileiros.

Objetivo da Plataforma

O sistema visa consolidar, processar e visualizar dados de diversas fontes oficiais (IBGE, DATASUS, FINBRA, SICONFI, INEP) em diferentes recortes espaciais — municípios, estados e regiões — oferecendo uma ferramenta analítica robusta para gestores públicos, pesquisadores e analistas de políticas públicas.

A plataforma combina técnicas de geoprocessamento com análise estatística para gerar insights acionáveis sobre o desempenho socioeconômico dos territórios brasileiros.

Módulos do Sistema

map

Inteligência Espacial

Mapa interativo com visualização de dados em polígonos (coroplético) e pontos, com drill-down por localidade.

bar_chart

Dashboard Analítico

Visualização comparativa guiada por dados estatísticos, com rankings, gráficos e ferramentas de comparação.

summarize

Gerador de Relatórios

Dossiê dinâmico com perfil completo de uma localidade, otimizado para impressão e exportação PDF.

database

Catálogo de Dados

Biblioteca interna com metadados, metodologias e acesso direto às fontes de dados oficiais utilizadas.
